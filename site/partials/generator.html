Kanji data

Two information fields are provided:

Meaning
Possible translations of the character into English terms.
Reading
All the various ways in which the character may be pronounced based on different contexts (excluding proper nouns, inclusion of which would make the database too large to submit!).

Study data

Four information fields are provided, generally available only for the Ministry of Education-recommended characters:

Frequency
A ranking of the frequency with which each of the characters are used (higher ranks are more frequent).
Stroke
The number of strokes used to draw the character.
JLPT Level
The level of the Japanese Language Proficiency Test under which the character falls.
Grade
The school grade in which native Japanese would learn the character.

Extra info

Four information fields are provided, generally available only for the Ministry of Education-recommended characters:

Henshall page
The page of Kenneth G. Henshall's 1988 character etymology dictionary (A Guide To Remembering Japanese Characters) under which the character may be found.
Henshall index
The index of the character under the same book.
Unicode
The Unicode point of the character.
JIS
The JIS point of the character.
